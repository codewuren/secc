#SECC

### Framework

[QT](https://www.qt.io/)

### Develop Tool

[QT Creator](https://www.qt.io/product/development-tools)

### Usage

Type:
```bash
secc --help
```
To know the usage.

You can quickly open a file with command:
```bash
secc a.cc
```
if you added it to you system `PATH`.

### Shortcut keys

| Key | Function |
| :-: | :-: |
| C-n | New File |
| C-f | Open File |
| C-s | Save File |
| C-S | Save as |
